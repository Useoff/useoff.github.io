Kemudian rederer dari Mali GLES dapat menggunakan Video Encoder drivers untuk memproses gambar dalam bentuk 2D/3D pada monitor atau layar.

Metode berikut digunakan saat riset:

1. Mendesign kustom Hardware Definition Languange (HDL) menggunakan Vivado IDE.
2. Kostumisasi linux kernel Video Encoder Driver's untuk merender gambar menggunakan Mali GPU secara langsung pada monitor atau layar.
3. Konfigurasi Linux Device Tree.
4. Melakukan testing fitur GLES dan driver Video Encoder menggunakan aplikasi yagears.

2021 Membagi process per CPU cores pada ZynqMP SOC (MYD-CZU3EG).

Memecah proses dari aplikasi untuk digunakan pada core CPU yang berbeda. ZynqMP SOC memiliki 2 Arm Cortex CPU: R (2 cores) dan A (4 cores).

Contoh Implementasi:

CPU Core Number Application
R5 0 Untuk memproses I/O, dan sinyal Analog
1 Memproses BUS dan interface data Serial
A53 0-1 Menjalankan Sistem Operasi Linux untuk menjalankan aplikasi monitoring shared data
2 None (Spare CPU)
3 None (Spare CPU)

2021 Implementasi LibMetal library untuk sharing data dalam area Share memory pada perangkat ZynqMP SOC (MYD-CZU3EG).

LibMetal library digunakan untuk membagikan data pada beberapa CPU proses ini juga dikenal sebagai Inter Processing Communication (IPC).

Digunakan untuk mendistribusikan data melalalui DDR memory. Metode ini berguna ketika menggunakan Inter-processing CPU (Split process) pada ZynqMP SOC.

2022 Pemprosesan data secara efektif menggunakan GPIO Interrupts Service Routine pada ZynqMP SOC.

Mengembangkan C / C++ wrapper class untuk mendapatkan permintaan GPIO interrupt dari Input pin.

GPIO Interrupt service routine digunakan pada C / C++ untuk mengolah data dari chip HI 6138. Chip ini digunakan untuk komunikasi data dengan protokol MIL-STD 1553.

2022 Mengembangkan C / C++ class wrapper untuk AXI UartLite dan Uart16550.

Membuat C / C++ class wrapper untuk memberikan fungisonalitas pengiriman dan penerimaan RAW data yang nantinya digunakan oleh aplikasi diatas layer class ini.

Penerapan fungsi pengiriman dan penerimaan data mengunakan metode poll dan interrupt agar dapat berfungsi dengan baik.
